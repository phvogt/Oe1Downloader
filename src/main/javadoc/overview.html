OE1 Downloader

Allows downloading of shows from the radio station OE1 via the 7 day archive.

Configuration

The program is a standalone application that is configured with the file conf/config.properties (see documentation of in the file).

The configuration file specifies the download folder and rules for every show that should be downloaded.

Process of downloading

The program works as follows:

  1. First all JSON data files are downloaded that contain all shows of the day (or starting from the current time 7 days ago).
  2. The JSON data are parsed and compared to the rules in conf/config.properties. A list of all shows to download generated.
  3. The file name for the MP3 for each show is determined by the rules in the config file.
  4. If there is not already a downloaded file for the show in the download directory, it is downloaded and stored there.
